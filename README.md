# 🧠 Responsive Quiz Application

A responsive quiz application built using **HTML**, **CSS**, and **Vanilla JavaScript** as part of my frontend development learning journey.

This project helped me understand how to build interactive web applications by working with the DOM, handling user events, managing application state, and creating responsive user interfaces.

---

## 🚀 Features

* Interactive multiple-choice quiz
* Responsive layout for desktop and mobile devices
* Dynamic question rendering
* Real-time score tracking
* Progress bar indicating quiz completion
* Visual feedback for correct and incorrect answers
* Personalized result message based on the final score
* Restart quiz functionality

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
